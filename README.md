## Installation

Install my-project with npm

```bash
  cd API
  npm install
  touch .env
  node server.js
  cd.. 
  cd client npm install
  npm run dev
```
.env file add DB_CONNECT = mongodb+srv://username:password@MONGOATLAS CLUSTER URL
